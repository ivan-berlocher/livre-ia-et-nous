### 22.7 Glossaire essentiel

**Les termes à connaître :**

| Terme | Définition simple |
|-------|------------------|
| **LLM** | Large Language Model — modèle de langage entraîné sur beaucoup de texte |
| **GPT** | Generative Pre-trained Transformer — architecture d'OpenAI |
| **Prompt** | L'instruction que vous donnez à l'IA |
| **Token** | Unité de texte (environ 0,75 mot en anglais) |
| **Fine-tuning** | Adaptation d'un modèle à un usage spécifique |
| **Hallucination** | Quand l'IA invente des informations |
| **Embedding** | Représentation numérique d'un texte |
| **RAG** | Retrieval-Augmented Generation — enrichir l'IA avec des documents |
| **Agent** | IA qui peut effectuer des actions, pas juste répondre |
| **Multimodal** | IA qui traite plusieurs types de données (texte, image, audio) |

### 22.9 Rester informé sans être submergé

**Stratégie de veille raisonnable :**

| Fréquence | Action |
|-----------|--------|
| **Quotidien** | Titre des actus (5 min max) |
| **Hebdomadaire** | 1-2 articles de fond |
| **Mensuel** | 1 rapport ou étude approfondie |
| **Trimestriel** | 1 livre ou formation |

**Éviter :**

- La FOMO (fear of missing out)
- Le doom scrolling d'actualités IA
- Les prédictions sensationnalistes
- Les débats toxiques sur les réseaux

**Privilégier :**

- Les sources de qualité
- Les analyses de fond
- Les témoignages concrets
- Les discussions constructives

### 22.10 Créer sa propre opinion

**Attention aux biais :**

| Biais | Comment l'éviter |
|-------|-----------------|
| **Techno-enthousiasme** | Cherchez les critiques |
| **Techno-pessimisme** | Cherchez les nuances positives |
| **Biais de confirmation** | Lisez ce qui vous dérange |
| **Effet de halo** | Séparez l'expertise d'un domaine d'un autre |
| **Récence** | Les dernières nouvelles ne sont pas les plus importantes |

**Questions à se poser :**

- Qui a intérêt à ce que je croie ceci ?
- Cette information est-elle vérifiable ?
- Quelle est l'autre version de l'histoire ?
- Est-ce que ça correspond à mon expérience ?

---

### Ce que vous pouvez faire

1. **Choisissez UNE ressource de chaque niveau.** Commencez petit.

2. **Définissez un temps de veille limité.** 30 minutes par semaine suffisent pour rester informé.

3. **Expérimentez vous-même.** Rien ne remplace l'usage direct.

4. **Discutez avec d'autres.** Les conversations enrichissent la compréhension.

5. **Restez humble.** Personne ne comprend tout. L'IA évolue trop vite.
